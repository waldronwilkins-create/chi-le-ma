# 🥢 吃了吗 — 项目文档

> 网址：https://waldronwilkins-create.github.io/chi-le-ma/
>
> 一个纯前端的互动网页，用于记录日常饮食和帮助决定每餐吃什么。

---

## 功能总览

### 📝 饮食记录（Tab 1）

**记录每一餐**

- 添加饮食记录：日期、餐别（早餐/午餐/晚餐/加餐）、食物名称
- **🏠 餐厅**：记录在哪吃的（2026-06-16 新增）
- **💰 人均价格**：记录每餐花费（2026-06-16 新增）
- **📸 食物照片**：拍照或从相册选择，Canvas 压缩至 800px JPEG 0.7 后存储（2026-06-17 新增）
- **📝 备注**：口味、份量等自由描述

**查看与管理**

- 按日期筛选 / 查看全部记录
- 按日期和餐别自动分组（早餐→午餐→晚餐→加餐）
- 每条记录显示缩略图（如有照片），点击放大灯箱
- 单条删除（联动删除对应照片）
- 📤 一键分享：弹出预览卡片，支持复制到剪贴板或系统原生分享

**统计条（紧凑型）**

- 今日记录数 / 本周记录数 / 总记录数 / 本周最爱食物 / **本月花费**（2026-06-17 新增）

---

### 🎲 食材挑选（Tab 2）

**选择食材**

- 50+ 种食材标签，点击切换选中状态
- 🔍 搜索过滤食材
- 🎲 **随机生成**：随机选一道菜，自动勾选其所有食材（2026-06-17 优化）
- ☑ 全选 / ✕ 清空

**菜肴匹配**

- **110 道中式菜肴数据库**（2026-06-17 从 60 道扩充至 110 道）
- 涵盖分类：家常菜、川菜、硬菜、主食、素菜、凉菜、海鲜、汤羹、快手菜、日韩料理、面点、甜品
- 根据已选食材智能匹配：绿色（已有）✅ / 红色（缺少）❌
- 完美匹配 ✨ / 部分匹配 🔍 标识
- 每道菜可一键「记入饮食」

**✏️ 自定义菜肴**（2026-06-17 新增）

- 添加自定义菜名、Emoji、分类、食材列表（逗号分隔）
- 自定义菜与内置菜合并参与匹配
- 可编辑和删除
- 自动校验菜名重复和食材非空

---

### 📊 数据统计（Tab 3 — 2026-06-17 新增）

**月度汇总**
- 每月总餐数 / 总花费 / 平均每餐价格

**餐厅排行**
- 按光顾次数降序排列，金银铜牌标识

**价格分布**
- < ¥20 / ¥20-50 / ¥50-100 / ≥ ¥100 区间统计，含占比

**存储用量**
- 照片占用空间实时统计，进度条显示
- ≥70% 黄色警告，≥90% 红色警告并阻止新上传

## 数据存储

所有数据存储在浏览器 **localStorage**，无需后端、无需注册：

| Key | 内容 | 格式 |
|---|---|---|
| `foodDiary` | 饮食记录数组 | JSON |
| `foodPhotos` | 照片字典 `{ entryId: base64 }` | JSON |
| `customDishes` | 自定义菜肴数组 | JSON |

**容量限制**：约 5MB，压缩后每张照片约 100-300KB，可存储 15-25 张照片。

## 技术实现

- **纯前端**：单文件 `index.html`，HTML + CSS + JavaScript，零依赖
- **适配手机**：响应式设计（600px 断点），支持 `capture="environment"` 调用摄像头
- **照片压缩**：Canvas 绘制，最长边 ≤800px，JPEG quality 0.7
- **分享**：Web Share API（系统分享）+ Clipboard API（复制）
- **部署**：GitHub Pages 自动部署
